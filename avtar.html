Avtar

Avtars are user profile pictures, usually fond in circular shapes. Avtars can be used for 'container' objects --

projects,spaces,repositories,etcetera -- to give them a recogonisable visual identity

Profile Avtar

Profile Avtars can be created using the class 'avtar', inside the img tag.

Sizes Avatar

Square Avtar

Adding a class 'square' to existing class, will generate a square shaped avatar.

Note : Square class wouldn't work if the image provided is svg.
